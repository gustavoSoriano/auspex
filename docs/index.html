Overview

What is Auspex?

Auspex is an AI agent that automates web browsers. Give it a URL and a natural language prompt — it decides whether HTTP scraping is enough or if it needs to launch a full browser.

⚡

Fast by Default

Tries HTTP/Cheerio first (100-500ms). Only launches browser when necessary.

🧠

AI Decisions

LLM chooses actions automatically. No manual scripting required.

🔒

Secure

Anti-SSRF, action whitelist, selector sanitization built-in.

🌐

Works Locally

Agentium support for 100% free, offline execution.

Architecture

How It Works

Three-tier strategy for optimal speed and capability.

1
Tier 1: HTTP/Cheerio

Download HTML with got-scraping, extract text/links/forms, send to LLM. If response is "done" → return without browser.

2
Tier 2: Stealth HTTP

With TLS fingerprinting and realistic headers for anti-bot bypass.

3
Tier 3: Playwright

Full Chromium browser with JavaScript support for complex interactions.

Getting Started

Quick Start

Basic Usage

import { Auspex } from "auspex";

const agent = new Auspex({
    llmApiKey: process.env.LLM_API_KEY!
});

const result = await agent.run({
    url: "https://news.ycombinator.com",
    prompt: "Return the title of the first article.",
});

console.log(result.data);
await agent.close();

100% Local (No API Key)

import { Auspex } from "auspex";

const agent = new Auspex({
    provider: "agentium",
    temperature: 0,
});

const result = await agent.run({
    url: "https://example.com",
    prompt: "What is the page title?",
});

console.log(result.data);
await agent.close();

With Typed Output (Zod)

import { z } from "zod";

const schema = z.object({
    title: z.string(),
    price: z.number(),
});

const result = await agent.run({
    url: "https://shop.example.com/product",
    prompt: "Extract title and price",
    schema,
});

// result.data is { title: string, price: number }

Capabilities

Available Actions

LLM can only execute whitelisted actions. Everything else is blocked.

Action Description
search Web search via SearXNG
click Click element
type Enter text in input
select Select dropdown option
pressKey Press keyboard key
hover Hover over element
goto Navigate to URL
wait Wait N milliseconds
scroll Scroll page
done Finish and return result
Safety

Security Features

Anti-SSRF

  • Only http:// and https:// protocols allowed
  • Blocks private IPs: 127.0.0.0/8, 10.0.0.0/8, 192.168.0.0/16, 172.16.0.0/12
  • Blocks cloud metadata: 169.254.169.254
  • DNS rebinding protection

Action Validation

  • Zod schema validation for all actions
  • Selector blacklist: javascript:, on*, <script>, data:
  • Max selector length: 500 chars

Loop Protection

  • Max iterations (default: 30)
  • Timeout (default: 120s)
  • Stuck detection: same action 3x in 9 iterations
Automation

Macros

Record successful runs and replay deterministically without LLM. 

import { macroToJsonString, replayMacroWithBrowser } from "auspex";

// Run with macro recording
const result = await agent.run({ url, prompt });
if (result.macro) {
    // Serialize
    const json = macroToJsonString(result.macro);

    // Replay without LLM (deterministic)
    const out = await replayMacroWithBrowser(result.macro, {
        actionDelayMs: 500,
    });
    // out.status === "ok" | "error"
}

Usage

Common Patterns

Multiple runs (reuse browser) ▼ 
const agent = new Auspex({ llmApiKey: "sk-..." });

const r1 = await agent.run({ url: "https://a.com", prompt: "..." });
const r2 = await agent.run({ url: "https://b.com", prompt: "..." });

await agent.close();
Web search integration ▼ 
const agent = new Auspex({
    llmApiKey: "sk-...",
    searxngUrl: "http://localhost:8080",
});

// No URL — searches first
const result = await agent.run({
    prompt: "Find iPhone 15 price on Apple site",
});
With browser pool ▼ 
const pool = new BrowserPool({ maxSize: 3 });
const agent = new Auspex({ llmApiKey: "sk-..." }, pool);
// ...
await agent.close();
await pool.close();
Event listeners ▼ 
agent.on("tier", (tier) => console.log("tier:", tier));
agent.on("action", (action, i) => console.log("action:", action.type));
agent.on("done", (result) => console.log("done:", result.status));
Best Practices

Tips

  • Be specific in prompts — "Go to pricing page, find Pro plan, return monthly price"
  • Use GPT-4o for complex tasks, gpt-4o-mini for simple ones
  • Set temperature: 0 for deterministic automation
  • Use allowedDomains in production for security
  • Reuse agent instances instead of creating new ones
  • Try Agentium for free local testing
